# statistic

（本题负责人：李博颖）

给出一个字符串（只包含小写字母），要求输出这个字符串中每个字母出现次数的统计图

### 输入

第一行一个整数 n 表示字符串的长度（n <= 100）

第二行一个长度为 n 的字符串

#### 样例输入

```
15
thisisabadapple
```



### 输出

如样例所示

#### 样例输出

```
 =                                                                            
 =                       =                    =        =                      
 =  =     =  =        =  =        =           =        =  =                   
------------------------------------------------------------------------------
 a  b  c  d  e  f  g  h  i  j  k  l  m  n  o  p  q  r  s  t  u  v  w  x  y  z 
```
